# (WIP) heroku-buildpack-ccv

A Heroku buildpack to [libccv](https://github.com/liuliu/ccv).

## Using

Point the `BUILDPACK_URL` config or add the following to your `.buildpacks`:

```
https://github.com/automata/heroku-buildpack-ccv.git
```

## Building

See `build.sh` for details on how to build your own libccv binary.
